# Feature Priorities

## Current Product Priority
- Make the N5 experience engaging enough for daily use

## High Priority Features
- Daily streaks and daily goals
- XP, levels, and lightweight rewards
- Better review loop for wrong answers
- Kana drills with fast repetition
- Vocabulary flashcards with stronger progress feedback
- Listening mini-practice for beginner words and phrases
- Grammar practice with immediate feedback
- Progress map showing what the learner has completed

## Good Additions
- Session summaries after each study block
- Personalized recommendations for what to study next
- Achievement badges tied to real learning milestones
- Friendly onboarding that quickly places the learner into the right flow

## Lower Priority
- Social features
- Competitive leaderboards
- Expanding to N4 before N5 feels polished
- Broad feature sprawl that does not improve retention or learning

## Feature Test
- A feature should improve one of these: motivation, clarity, repetition, retention, or habit
