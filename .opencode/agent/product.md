# Product

## App
- Nihongo learning app for Japanese beginners
- Current live focus: JLPT N5
- Existing app already includes SRS, progress tracking, onboarding, review, challenge, and profile flows

## Users
- Absolute beginners learning kana, vocab, kanji, and basic grammar
- Mobile-first learners studying in short sessions
- Users who need visible progress and frequent motivation

## Goal
- Make early Japanese study feel fun, clear, and rewarding
- Increase daily return rate before expanding scope
- Help users build confidence fast instead of feeling overwhelmed

## Current Scope
- N5 learning experience is the priority
- N4 can exist in data model or roadmap, but should not distract from polishing N5

## Product Principle
- Prioritize beginner retention over feature count
- Prefer small, high-frequency wins over large complex systems
- Every new feature should support learning, motivation, or habit formation

## Success Signals
- Users can start studying quickly
- Users understand what to do next
- Users see progress after each session
- Users feel encouraged to come back tomorrow
